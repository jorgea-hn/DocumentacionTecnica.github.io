¿Que es HTML?

HTML significa lenguaje de marcado de hipertexto, este sirve para crear paginas web, y permite describir la estructura de la pagina web.

HTML basico

Todos los documentos HTML deben comenzar con la declaracion de tipo de documento <!DOCTYPE html>, y comienza con una etiqueta < html> y termina con < html>, dentro de estas etiquetas existen dos etiquetas mas las cuales son: <head> donde van el nombre de la pagina, autor, palabras claves, el enlace al documento de estilos, entre otras cosas y un <body> que es la parte visible del documento HTML, aqui va toda la informacion que queremos que vea el usuario.

Existen mas etiquetas que tienen diversas funciones por ejemplo,< h1> que es para titulos, la etiqueta <p> que es para colocar parrafos, <br> para dar un espacio abajo.

Encabezados

Los encabezados son titulos o subtitulos que se desean mostrar en la pagina web, y se definen con las etiquetas que van desde la < h1> hasta la < h6> donde la < h1> es la de mayor importacia o el titulo principal y la < h6> la de menor importancia.

Estilos css

El atributo Style se usa para agregar estilos a un elemento, tal como el color, el tamaño, la tipografia, entre otras cosas.

Esta se puede usar de diferentes maneras

  • En linea, mediante el uso del atributo style dentro de los elementos del html
  • Interno, mediante el uso de la etiqueta < style> en la etiqueta < head>
  • Externo, mediante el uso de la etiqueta < link> para vincular el archivo html a un archivo css externo
Comentarios

Para realizar comentarios en html solo se debe colocar la etiquetas <!-- al principio y --> al final del texto

Enlaces

Estos enlaces son tambien conocidos como hipervinculos, al hacer click te redirige a otra pagina la etiqueta para esta funcion es < a>

Listas

Las listas permiten a los desarrolladores y existen dos tipos de listas, las ordenadas y las desordenadas, las listas desordenadas comienzan con la etiqueta <ul> y cada elemento tiene sus item que tienen la etiqueta <li>, mientras que las listas ordenadas tienen la etiqueta <ol>

Ejemplo lista desordenada

  • queso
  • pan
  • lechuga
  • tomate
  • carne
Id y Clases

Las id se utilizan como una identificacion unica de un elemento html, este atributo se utiliza en las hojas de estilo css y en javascript para poder manipular elementos y para llamarlos se utiliza el simbolo "#".

Las clases se utilizan para identificar a varios elementos que poseen algo en comun, y se utilizan tambien en hojas de estilo y en javascript, para llamarlos se utiliza el simbolo "."
